## 技术栈

- **框架**: React 19 + TypeScript
- **路由**: React Router v7
- **状态管理**: Zustand + React Query
- **图表**: Recharts
- **样式**: Tailwind CSS
- **图标**: Lucide React
- **日期处理**: date-fns
- **表单验证**: Zod
